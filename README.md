# Resturant-basic-ui
Priyansu Restro is a modern and responsive restaurant website designed to represent a real-world dining business online. The website focuses on clean design, smooth user experience, and attractive food presentation.
      Features
Modern navigation bar with branding
Hero section with restaurant identity
Story section explaining the restaurant concept
Signature Dishes section with food images and prices
Image gallery showcasing restaurant moments
Visit section with location, contact, and timing
Fully responsive design (mobile & desktop friendly)
    Technologies Used
HTML5
CSS3
Unsplash images for food visuals
No frameworks or libraries are used, keeping the project lightweight and easy to understand.
